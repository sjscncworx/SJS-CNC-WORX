# SJS CNC WORX Website

A professional website for SJS CNC WORX - a leading CNC machining and fabrication company.

## 🎨 Design Features

- **Modern Industrial Aesthetic**: Sleek neon lime green (#00FF00) on black background
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Layout**: Easy navigation across all pages
- **Contact Integration**: Built-in contact form with email functionality
- **Optimized Performance**: Fast loading, clean code

## 📄 Pages Included

1. **Home (index.html)** - Landing page with hero section and services overview
2. **About (about.html)** - Company information, mission, and director details
3. **Services (services.html)** - Detailed service descriptions and offerings
4. **Gallery (gallery.html)** - Project showcase and statistics
5. **Contact (contact.html)** - Contact form and company contact information

## 🚀 How to Use

### Local Testing
1. Open any HTML file in your web browser
2. Navigate using the menu links
3. Test the contact form functionality

### Hosting Your Website
1. **Domain Connection**: Contact your domain registrar to point your domain to your hosting provider
2. **Upload Files**: Upload all files to your web hosting server via FTP
3. **Email Setup**: Configure email forwarding for the contact form
4. **SSL Certificate**: Enable HTTPS (most hosts provide free SSL)

## 🔧 Customization Tips

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-green: #00FF00;    /* Change this color */
    --dark-bg: #0a0a0a;          /* Change background */
    --text-light: #ffffff;        /* Change text color */
}
```

### Modify Content
Each HTML file has clear sections:
- Navigation - Update links here
- Hero/Content - Main page content
- Footer - Contact info and links

### Add More Pages
1. Create new HTML file with similar structure
2. Update navigation in all files to include new page
3. Follow the existing styling pattern
